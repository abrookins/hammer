Hammer
======

A library used to generate JSON Schemas from Colander schemas.

Note: This library is under active development and not yet ready for use.


Working
=======

SchemaTypes:

    - colander.Int
    - colander.Integer
    - colander.Float
    - colander.String
    - colander.Str
    - colander.Bool
    - colander.Dated
    - colander.DateTime
    - colander.Time

Schemas:

    - colander.Schema
    - colander.MappingSchema
    - colander.Mapping
    - colander.Sequence
    - colander.Tuple

Validators:

    - colander.Regex
    - colander.Email
    - colander.Range
    - colander.Length
